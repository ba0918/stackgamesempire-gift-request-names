# stackgamesempire-gift-request-names

## Installation
```bash
npm install @ba0918/stackgamesempire-gift-request-names
```

## Usage
```
stackgamesempire-gift-request-names <Gift URL> --steamkey <Your Steam API Key> --steamid <Your SteamID> --output <path>
```

## Development
### Setup
```bash
yarn install
```

### Build
```bash
yarn run build
```
